# Nova Space Invaders

Prototype web jouable de type Space Invaders.

## Lancer
- Ouvrir `index.html` dans un navigateur moderne

## Fichiers
- `index.html` : structure de la page
- `styles.css` : interface et mise en page
- `game.js` : boucle de jeu et logique
- `SPEC.md` : spec snapshot V1
- `TEST_PLAN.md` : plan de validation
- `DECISION_LOG.md` : decisions d'architecture
